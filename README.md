# Vostok.Hercules.Client.Abstractions

Facade with main Hercules client interfaces and data model.

Interfaces:
- `IHerculesSink`
- `IHerculesGateClient`
- `IHerculesStreamClient` and `IHerculesStreamClient<T>`
- `IHerculesTimelineClient`
- `IHerculesManagementClient`
